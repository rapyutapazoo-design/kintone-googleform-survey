# Kintone × Google Forms アンケートシステム

Kintoneを「アンケートの入り口」、Google Formを「アンケート本体」として組み合わせ、
両者の接続を**常設のGoogle Apps Script（GAS）1本**で全自動化するシステムです。

管理者はGoogle Driveの指定フォルダにフォームを作るだけ。ID・URLのコピペ、
スクリプトの編集、Webhookやトリガーの設定作業は一切不要です。

## 特徴

- **コピペゼロ運用** — フォルダ監視GASがフォームを検知し、Kintoneへの登録・トリガー設置まで自動実行
- **回答済みの即時反映** — 送信トリガーで回答者を記録し、Kintone上のボタンが「✅ 回答済み」に切り替わる（5分おきの補修同期付きで自己修復）
- **回答データはGoogleに集約** — Kintoneには回答内容を保存せず、回答の確認・集計はFormの回答タブで完結。回答者には控えメールを自動送信
- **受付終了も自動** — Kintoneのステータスを「終了」にするだけで、フォームの受付停止と不要トリガーの掃除まで自動
- **Webhook・Webアプリデプロイ不要** — 可動部は「5分おきの時間トリガー1個」のみ

## アーキテクチャ

```
┌─ Google Drive「アンケート」フォルダ ─────────────┐
│  マスターフォーム（フォルダ外・コピー元）             │
│  各アンケートフォーム（管理者がコピー作成）            │
└──────────┬───────────────────────┘
           │ 監視（5分おき・時間トリガー1個）
┌─ 常設GAS 1本 ─────────────────────────┐
│ ① 新フォーム検知 → 初期設定 + トリガー設置          │
│    + Kintoneへレコード自動作成                     │
│ ② 回答送信時 → 回答者コードを即時記録               │
│    （5分おきの差分同期で取りこぼしを自動補修）        │
│ ③ ステータス「終了」検知 → フォーム受付の自動停止     │
└──────────┬───────────────────────┘
           │ REST API（APIトークン）
┌─ Kintone アンケート管理アプリ ─────────────────┐
│ 一覧・詳細・モバイルカードにボタンを表示              │
│ 📝 回答する ／ ✅ 回答済み ／ ⛔ 受付終了            │
└──────────────────────────────────┘
```

## 運用フロー（毎回）

```
① マスターフォームを「コピーを作成」→ 監視フォルダへ → 質問編集
② 最大5分待つ（自動でKintoneに登録される）
③ Kintoneでステータスを「受付中」に変更 → 公開
④ 終了時はステータスを「終了」に変更するだけ
```

## 動作要件

- Kintone（cybozu.com）環境とアプリ管理権限
- Google アカウント（Drive / Forms / Apps Script）
- 回答者はGoogleアカウント不要（フォームはログイン不要設定で運用）

## 制約・既知の事項

- GASのフォーム送信トリガーはプロジェクトあたり約20個が上限のため、同時開催アンケート数に上限があります（終了したアンケートのトリガーは自動削除されます）
- 「回答のコピーを回答者に送信」はAPIから設定できないため、マスターフォームからのコピー作成が運用ルールです
- 回答済み判定はフォーム内のユーザーID質問（事前入力）に依存します。回答者がIDを書き換えると判定できません
