# direct-link/ — QR直リンク版（中継ページなし）

QRの飛び先を **GAS中継ページ（Webアプリ）ではなくGoogleフォームの直リンク**にする版です。
ルートの中継版（`gas/survey-gas.js`）は将来戻せるようそのまま残してあり、本フォルダはその代替実装です。

## 中継版との違い

| 項目 | 中継版（`gas/`） | 直リンク版（このフォルダ） |
|---|---|---|
| QRの飛び先 | GAS中継ページ `…/exec?fid=…` | Googleフォーム直リンク（`form_qr_url`） |
| 警告バナー | 出る | **出ない** ✅ |
| マルチログイン | 開けないことがある | **確実に開ける** ✅ |
| QRでの回答前ブロック | あり | **なし**（重複は集計時に排除） |
| ウェブアプリのデプロイ | 必要 | **不要**（時間トリガーのみ） |
| 回答経路の区別 | あり（中継が経路=QRを付与） | あり（`form_qr_url` に経路=QRを事前入力） |

## 共有・差分のあるファイル

| ファイル | 扱い |
|---|---|
| `direct-link/survey-gas.js` | 本フォルダの直リンク版GAS（中継版から `doGet` 等を撤去し `form_qr_url` 生成を追加） |
| `kintone/survey-portal.js`（ルート） | **共有・無変更**（Kintone導線は中継ページに依存しないため両版共通） |
| `direct-link/セットアップ手順書.md` | 直リンク版の手順（中継版との差分中心） |

## 中継版へ戻す場合

1. GASエディタのコードを `gas/survey-gas.js`（中継版）に貼り替える
2. ウェブアプリとして再デプロイ（実行ユーザー: 自分／アクセス: 全員）
3. QRを中継ページURL（`…/exec?fid=…`）で作り直す
4. （任意）`form_qr_url` フィールドは未使用になるが残しておいてよい
